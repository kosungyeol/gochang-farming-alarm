# 🌾 고창 농업 알리미

고창군 농업인을 위한 보조사업 안내 및 알림 앱

## 📱 앱 소개

**고창 농업 알리미**는 고창군 농업인(특히 60세 이상)을 대상으로 한 보조사업 안내 및 알림 서비스입니다.

### 주요 특징
- 🔊 **TTS 음성 안내**: 성별/속도 조절 가능한 음성 안내
- 📱 **간단한 UI**: 고령자 친화적인 큰 버튼과 글자
- 🔔 **스마트 알림**: 사용자 활동 패턴 기반 최적 시간 알림
- 🎨 **접근성 최적화**: 한지 느낌 디자인, 고대비 테마 지원
- 🌐 **관리자 시스템**: 엑셀 업로드로 쉬운 사업 관리

## 🖥️ 화면 구성

1. **개인정보 입력화면** - 최초 가입시 정보 입력
2. **메인 홈화면** - 새 알림, 이달의 사업, 내 관심사업
3. **관심사업 선택화면** - 월별 사업 관리
4. **알림화면** - 사업별 상세 안내 및 TTS 지원
5. **설정화면** - 개인정보, 알림설정, 화면설정

## 🚀 시작하기

### 필수 조건
- Node.js 16+ 
- React Native CLI
- Android Studio (Android 개발시)
- Xcode (iOS 개발시)

### 설치 및 실행

```bash
# 의존성 설치
npm install

# iOS Pods 설치 (iOS만)
cd ios && pod install && cd ..

# Android 실행
npm run android

# iOS 실행
npm run ios

# Metro 서버 시작
npm start
```

## 🎨 디자인 시스템

### 색상 테마
- **한지 테마** (기본): 크림색/베이지색 배경, 진한 갈색 텍스트
- **밝은 테마**: 흰색 배경, 검은색 텍스트
- **어두운 테마**: 검은색 배경, 흰색 텍스트
- **고대비 테마**: 접근성을 위한 고대비 색상

### 폰트 크기
- 작게: 16px
- 보통: 18px (기본)
- 크게: 22px
- 매우 크게: 26px

## 🔧 주요 기능

### TTS (Text-to-Speech)
- 한국어 음성 지원
- 남성/여성 목소리 선택
- 4단계 속도 조절 (매우 느림 ~ 빠름)
- 화면별 자동 안내

### 푸시 알림
- Firebase Cloud Messaging 기반
- 사용자 활동 패턴 분석
- 최적 시간 자동 발송
- 중복 알림 방지

### 접근성
- 큰 터치 영역 (최소 44px)
- 고대비 색상 옵션
- 스크린 리더 지원
- 간단한 네비게이션

## 📊 관리자 시스템

### 기능
- 📈 실시간 사용자 통계
- 📄 엑셀로 사업 정보 업로드
- 🔄 개별 사업 수정/삭제
- 📱 알림 발송 관리
- 📊 상세 분석 리포트

- TTS 스크립트, 알림 주기
- 진행 안내 일정 등
